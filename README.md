# Intelligent Fraud Detection and Risk Scoring System

This project detects fraudulent financial transactions using machine learning and provides risk scores through a deployed API.

## Features
- Fraud detection using Logistic Regression
- Risk scoring based on probability
- Real-time API using Flask
- Deployed on Render

## Tech Stack
Python, Pandas, Scikit-learn, Flask

## API Endpoint
POST /predict
